# Esoteric-Special-Weapons

A mod featuring several weapons and one new location. These weapons have no real particular order to them, I just felt like they were interesting to have in the game

-7 New Launchers
-6 new grenades
-6 new melee weapons
-14 new pistols
-29 new rifles
-5 new SMGs
-1 new trap
-1 new Location
